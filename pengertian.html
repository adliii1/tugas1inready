Pengertian HTML

HTML adalah singkatan dari Hypertext Markup Language, yaitu bahasa markah standar untuk membuat dan menyusun halaman dan aplikasi web. HTML menggunakan tag untuk menandai elemen-elemen dalam dokumen, seperti teks, gambar, tabel, dan tautan. HTML adalah bahasa pemrograman yang sangat penting untuk membuat situs web. Tanpa HTML, situs web tidak akan dapat ditampilkan di browser.
