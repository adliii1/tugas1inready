Structur HTML

Struktur utama html terdiri dari tag html, tag head, tag body, dan tag footer, berikut penjelasannya

  1. Tag HTML

    Tag HTML adalah tanda atau simbol yang digunakan untuk menandai elemen-elemen dalam dokumen HTML. Tag HTML selalu ditulis berpasangan, yaitu tag pembuka dan tag penutup.

  2. Tag Head

    Tag head dalam HTML adalah container untuk metadata (data tentang data) dan diletakkan di antara tag html dan tag body. Metadata adalah data tentang dokumen HTML yang tidak ditampilkan di halaman web.

  3. Tag Body

    Tag body dalam HTML adalah container untuk konten utama dari dokumen HTML, dan diletakkan di antara tag html dan tag html. Konten utama dari dokumen HTML adalah konten yang akan ditampilkan di halaman web, seperti teks, gambar, tabel, dan tautan.

  4. Tag Footer

    Tag footer dalam HTML adalah container untuk konten tambahan dari dokumen HTML, dan diletakkan di antara tag body dan tag body. Konten tambahan dari dokumen HTML adalah konten yang biasanya diletakkan di bagian bawah halaman web, seperti informasi tentang hak cipta, informasi kontak, atau tautan ke media sosial.
